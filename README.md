# Name: Credit Card Spending Habits In India Analysis

## Description
The project aims to analyse data on spending in India to identify trends, key factors.

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Scipy.Stats
- Tableau
